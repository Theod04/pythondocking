#  Εργαλείο Μοριακής Πρόσδεσης 

Μια εφαρμογή Python για την εκτέλεση Μοριακής Πρόσδεσης (Molecular Docking) χρησιμοποιώντας το λογισμικό **AutoDock Vina** μέσω ενός φιλικού γραφικού περιβάλλοντος (Streamlit).

Η εφαρμογή αναπτύχθηκε στο πλαίσιο ακαδημαϊκής έρευνας για τη μελέτη φαρμακευτικών στόχων που σχετίζονται με νευροεκφυλιστικές ασθένειες (Alzheimer's, Parkinson's).

##  Δυνατότητες

* **Αυτόματη Μετατροπή:** Μετατρέπει κωδικούς **SMILES** (φάρμακα) απευθείας σε μορφή `.pdbqt` έτοιμη για docking.
* **Στοχευμένη Πρόσδεση (Targeted Docking):** Επιτρέπει τον ακριβή καθορισμό του χώρου αναζήτησης (Grid Box) με συντεταγμένες X, Y, Z.
* **Υπολογισμός Ενέργειας:** Εκτελεί το AutoDock Vina και υπολογίζει την ενέργεια σύνδεσης (Binding Affinity σε kcal/mol).
* **3D Οπτικοποίηση:** Εμφανίζει διαδραστικά το σύμπλοκο πρωτεΐνης-φαρμάκου αμέσως μετά την ανάλυση.
* **Εξαγωγή Αρχείων:** Δυνατότητα λήψης των αποτελεσμάτων (Docked Ligand & Complex).

##  Απαιτήσεις Συστήματος

Η εφαρμογή απαιτεί **Python 3.8+** και τις παρακάτω βιβλιοθήκες:

* streamlit
* py3Dmol
* rdkit
* meeko

*(Σημείωση: Το εκτελέσιμο `vina.exe` συμπεριλαμβάνεται στο αποθετήριο για περιβάλλον Windows).*

##  Εγκατάσταση & Εκτέλεση

1.  **Κλωνοποίηση αποθετηρίου (Clone):**
    ```bash
    git clone [https://github.com/ΤΟ_ONOMA_ΣΟΥ/ionian-docking-tool.git](https://github.com/ΤΟ_ONOMA_ΣΟΥ/ionian-docking-tool.git)
    cd ionian-docking-tool
    ```

2.  **Εγκατάσταση βιβλιοθηκών:**
    ```bash
    pip install -r requirements.txt
    ```

3.  **Εκτέλεση εφαρμογής:**
    ```bash
    streamlit run app.py
    ```

## Μεθοδολογία & Στόχοι

Η εφαρμογή χρησιμοποιείται για τη μελέτη πρόσδεσης μικρών μορίων σε πρωτεΐνες-στόχους κλειδιά για τη νόσο Alzheimer. Οι συντεταγμένες του Grid Box καθορίζονται με βάση τη βιβλιογραφία (Co-crystallized ligands) από το Protein Data Bank (PDB).

| Πρωτεΐνη (PDB Code) | Βιολογικός Ρόλος / Θεωρία | Στόχος Φαρμάκου |
| :--- | :--- | :--- |
| **AChE (4EY7)** | Χολινεργική Υπόθεση | Καταλυτική Χαράδρα (Catalytic Gorge) |
| **NMDA (7SAD)** | Διεγερσιτοξικότητα | Κανάλι Ιόντων (Pore Blocker) |
| **MAO-B (2V5Z)** | Οξειδωτικό Στρες | Υδρόφοβη Κοιλότητα (Hydrophobic Pocket) |
| **MARK4 (5ES1)** | Υπόθεση Tau | Ενεργό Κέντρο Κινάσης |
| **GLP-1R (7S15)** | Μεταβολική Υπόθεση | Διαμεμβρανική Περιοχή |

##  Σημαντική Σημείωση

Για την ορθή λειτουργία του Docking, απαιτείται η εισαγωγή των σωστών συντεταγμένων (Center & Size) που έχουν προκύψει από ανάλυση της κρυσταλλικής δομής με εργαλεία όπως το **UCSF Chimera**.
